📲 Telegram Alert System

AI-powered alert system that delivers real-time business insights directly to Telegram, turning data into immediate action.

OVERVIEW

📊 What this system does

This system automatically collects customer reviews, analyzes them using AI, and sends structured business insights directly to Telegram.

Instead of checking dashboards or reading reports, business owners receive instant, actionable intelligence in real-time on their phone.

PROCESS

⚙️ How it works

  • Collects customer reviews using Apify (Google Maps scraping)
  • Processes and aggregates review data
  • Uses AI to generate structured insights
  • Formats output for mobile readability
  • Sends real-time alerts via Telegram bot
AUTOMATION

🔄 Make.com Scenario

Fully automated workflow that runs without manual input and delivers insights instantly.

OUTPUT

📲 Example Output

The system generates structured Telegram messages including:

  • Summary of customer sentiment
  • Top positive feedback
  • Key problems detected
  • Actionable improvement steps
  • Business impact insights
BUSINESS VALUE

💡 Why this matters

  • No need to check dashboards or reports
  • Instant awareness of customer issues
  • Faster decision making
  • Prevents negative reviews from scaling
  • Turns feedback into direct action
ROI

💰 What businesses save

  • Eliminates manual review analysis
  • Reduces response time from days to minutes
  • Prevents revenue loss from bad reviews
  • Replaces manual reporting workflows
TECH

🧠 Tech Stack

  • Make.com automation
  • Apify (data collection)
  • Google Gemini AI (analysis)
  • Telegram Bot API
